Carro seguidor con circuito integrado

Este es un carro desarrollado con Arduino tanto como el Hardware y Software (el hardware de un Arduino Uno y su propio lenguaje de programacion). Este carrito tiene la particularidad que tiene circuito integrado, eso quiere decir que desde un principio a la hora de hacer la programacion del carro se le hace la respectiva ruta que hara el carro una vez dessarrollado y ensamblado
